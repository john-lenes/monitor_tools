Como usar

  1. Abra o Sankhya ERP na aba desejada do Chrome.
  2. Nomeie a sessão (ex: “Inclução de NF”) e clique ▶ Iniciar.
  3. Execute a ação que deseja analisar no Sankhya — todas as chamadas HTTP serão capturadas.
  4. Clique ■ Finalizar para encerrar a captura após a ação.
  5. Clique 📋 Ver Relatório Completo para ver análise detalhada, sugestões e exportações.

Dicas

  • Nomeie a sessão com a ação testada para identificar relatórios exportados.
  • Use “Apenas Prioritárias” para focar em gargalos e erros críticos.
  • Abra o DevTools do Chrome para captura mais precisa do corpo da resposta.
  • Exporte o TXT para compartilhar com o time de backend.
